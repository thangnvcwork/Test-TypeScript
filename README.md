# Vertical Scroll Video Feed

## Công nghệ sử dụng

- Next.js
- TypeScript
- Tailwind CSS
- React Hooks
- Intersection Observer API

## Chức năng chính

- Hiển thị danh sách video dạng cuộn dọc.
- Mỗi video chiếm toàn bộ màn hình.
- Click vào video để Play/Pause.
- Video tự động phát khi nằm trong viewport.
- Video tự động dừng khi bị cuộn qua.
- Nút Like có thể đổi màu và tăng/giảm số lượng like.
- Responsive: desktop hiển thị sidebar, mobile hiển thị bottom navigation.
